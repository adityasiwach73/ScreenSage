# 📱 ScreenSage

**ScreenSage** is a reward-based parental control Android application built using **Kotlin** and **Firebase**. It helps parents monitor their child's smartphone usage while encouraging healthier digital habits through a token-based reward system.

Instead of relying only on restrictions, ScreenSage motivates students to reduce unnecessary screen time by rewarding them with tokens. These tokens can be redeemed for additional usage time on selected applications, creating a balanced and positive approach to digital well-being.

---

## ✨ Features

- 👨‍👩‍👧 Separate Parent and Student Dashboards
- 📊 Daily Screen Time Monitoring
- 📱 App-wise Usage Tracking
- 🪙 Reward Token System
- ⏳ Redeem Tokens for Extra App Usage
- 🔐 Firebase Authentication
- ☁️ Firebase Database Integration
- 📈 Encourages Healthy Digital Habits

## 💡 Project Overview

ScreenSage was developed to address excessive smartphone usage among students. The application provides parents with insights into their child's screen time and application usage while encouraging responsible device usage through a reward-based token system.

Parents can monitor activity from their dashboard, while students can earn tokens by reducing their daily screen time. Earned tokens can later be exchanged for additional usage time on selected applications.
